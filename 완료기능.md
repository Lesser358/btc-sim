# 완료 기능 전체 목록

| 기능 | 비고 |
|------|------|
| 차트 | Lightweight Charts v4.2.3, 캔들+볼륨 |
| 시간봉 | 1m~1M 전체 (초봉 제외) |
| 가격선 | 진입/청산/TP/SL/지정가 — 실선·라벨 |
| 무한 스크롤 | 초기 1000봉 로드, 좌측 스크롤 시 1000봉씩 추가(`loadOlder`) |
| 실시간 | REST 폴링 backbone (WS 보조), 1초 갱신 |
| 그리기 도구 | 추세선·수평/수직선·사각형·피보나치·롱숏 포지션(3레벨 R/R) |
| 그리기 저장 | 심볼별 localStorage (`btcSimDrawings_<sym>`) |
| 거래 | 롱/숏, 격리/교차, 2~125x, 시장가/지정가, 수수료, 펀딩비, 강제청산 |
| TP/SL | 진입 시 설정 + 보유 중 가격/ROE% 모드 수정, 발동 시 전량 청산 |
| 청산 | % / USDT직접입력 / 전체 |
| 포지션탭 | 표(매초 라이브)와 입력칸(`posCtlBox`) 분리 → 입력 중 값 유지 |
| 평행채널 | `channel` 타입, 2단계 드래그(기준선→너비 클릭 확정), 연장선+채움 |
| 화살표 | `arrow` 타입, 드래그로 끝점에 화살촉 렌더 |
| 텍스트 | `text` 타입, 클릭 후 플로팅 입력창으로 레이블 입력 |
| 봉마감 카운트다운 | 각 시간봉 버튼 아래 남은시간 실시간 표시 (MM:SS / HH:MM:SS / N일 HH:MM) |
| TP/SL 봉 고/저가 체크 | `mergeCandle`에서 `checkPending(c.high, c.low)` — 순간 터치도 감지 |
| 계좌 초기화 부분 보존 | `resetAccount` — 잔액·포지션·내역만 리셋, 지표·레버리지·마진모드·알람·심볼·시간봉 유지 |
| 지표 시스템 | 지표+ 버튼 → 검색 모달 → 칩 표시 → 클릭 시 설정 패널 (기간·색상·굵기) |
| 오버레이 지표 | SMA·EMA·WMA·Hull MA·볼린저밴드 — 메인 차트에 렌더, 실시간 마지막봉 업데이트 |
| 오실레이터 지표 | RSI·MACD·스토캐스틱·ATR — 차트 하단 별도 패널, 메인 차트와 스크롤 동기화 |
| 지표 설정값 고정 | `applyIndSettings` 시 `S.indDefaults[type]`에 저장 → 같은 타입 재추가 시 마지막 값으로 시작 |
| 알람 기능 | 차트 툴바에 「알람」버튼 → 패널(top-right 고정) → Hull MA 근접 알람 (USDT 거리·방향·마감봉 기준·볼륨 슬라이더) |
| 알람 오디오 개선 | Web Audio API(AudioContext+AudioBuffer) — 배경탭 재생, GainNode 3× 증폭, 소리 테스트 버튼 |
| TP/SL 감지 버그 수정 | WS 불안정 시 `markPriceUpdate` → `liveTick` → `checkPending` 경로로 봉 고/저 체크 보장 |
| 차트 부드러움 | 200ms 인터벌 — `lastPrice`로 forming candle 매초 5회 갱신 |
| 통계 버그 수정 | 최고수익/최대손실 양수/음수일 때만 표시, 해당 없으면 `-` |
| 미체결 주문 TP/SL 표기 | 미체결 탭 테이블에 TP·SL 가격 컬럼 추가, ROE% 모드도 절대가로 변환 표시 |
| 미체결 주문 TP/SL 수정 | 지정가 주문 행 「수정」버튼 → 팝업 패널에서 TP/SL 가격·청산량 수정 후 저장 |
| 부분 청산 | TP/SL 청산량 드롭다운 — `%` / `USDT` / `코인` 3가지 단위 선택, 주문폼+포지션탭 모두 지원 |
| 용어 통일 | 익절→TP, 손절→SL (주문탭·포지션탭·미체결탭 전체) |
| 거래내역 | `MM/DD HH:MM:SS` 날짜 컬럼 + ROE% 컬럼 (강제청산 -100% 고정) |
| 포지션탭 진입시간 | 방향 옆에 `MM/DD HH:MM` 형식 진입시간 컬럼 |
