============================================
 BTC 선물 모의투자 (바이낸스 기준) 사용설명서
============================================

[실행 방법]
- index.html 파일을 더블클릭하면 기본 인터넷 브라우저에서 열립니다.
  (크롬 / 엣지 권장)
- 설치할 것 없음. 인터넷만 연결되어 있으면 됩니다.
  (바이낸스 선물 실시간 시세를 가져옵니다)
- 앱 안에서 ? 버튼을 누르면 더 자세한 도움말을 볼 수 있습니다.

----------------------------------------------
[거래 코인 종목]
----------------------------------------------
- 헤더 왼쪽 코인 버튼을 클릭해 종목을 전환할 수 있습니다.
- 지원 종목: BTC, ETH, SOL, BNB, XRP, DOGE, ADA, AVAX (8종목)
- 종목별 포지션·거래 내역이 각각 기록됩니다.

----------------------------------------------
[시간봉 선택]
----------------------------------------------
- 차트 상단 시간봉 버튼을 누르면 바로 전환됩니다.
- ▾ 버튼으로 더 많은 시간봉을 드롭다운에서 선택할 수 있습니다.
- ★ 버튼으로 자주 쓰는 시간봉을 즐겨찾기에 추가/제거할 수 있습니다.
  (즐겨찾기는 자동 저장되어 다음 실행 시 유지됩니다)

----------------------------------------------
[기본 거래 방법]
----------------------------------------------
1. 오른쪽에서 '격리/교차' 마진 모드를 고릅니다.
2. 레버리지를 2배~125배 사이로 조절합니다.
3. '시장가' 또는 '지정가' 주문을 선택합니다.
4. '증거금(투입 금액)'을 입력하거나 10/25/50/75/100% 버튼을 누릅니다.
   - 입력 즉시 주문 수량 / 포지션 가치 / 예상 청산가가 미리 보입니다.
5. (선택) TP(익절가), SL(손절가)를 넣으면 그 가격에 자동 종료됩니다.
6. 초록 '롱(매수)' 또는 빨강 '숏(매도)' 버튼을 누릅니다.

----------------------------------------------
[TP / SL (익절·손절) 설정]
----------------------------------------------
- 주문 전 TP/SL 입력란에 가격을 미리 설정할 수 있습니다.
- '가격' / 'ROE%' 드롭다운으로 설정 방식을 전환할 수 있습니다.
  · 가격: 특정 가격에서 종료
  · ROE%: 수익률(%) 기준으로 종료 (예: ROE 50% → 목표 수익 50%에서 익절)
- TP/SL은 '봉 고가/저가' 기준으로 감지됩니다.
  (체결은 해당 봉 마감 시 설정가로 적용)
- 포지션 진입 후 차트 위 TP/SL 가격선을 마우스로 위아래 드래그하면
  가격을 실시간으로 수정할 수 있습니다. (드래그 후 확인 모달 표시)

----------------------------------------------
[포지션 청산]
----------------------------------------------
- 포지션 탭 하단 '청산' 버튼으로 전체 종료합니다.
- 일부만 청산: 수량 입력 후 % 또는 USDT 단위를 드롭다운으로 선택 가능.
  예) 50% 입력 → 포지션의 절반만 청산

----------------------------------------------
[미체결 주문 탭]
----------------------------------------------
- 하단 '미체결 주문' 탭에서 지정가 주문 목록을 확인할 수 있습니다.
- 각 주문의 취소 버튼으로 개별 취소 가능.

----------------------------------------------
[거래 내역 탭]
----------------------------------------------
- 하단 '거래 내역' 탭에서 과거 체결 내역을 확인합니다.
- 종목, 방향, 진입가, 종료가, 실현손익, ROE%, 날짜가 표시됩니다.

----------------------------------------------
[지표 (차트 지표 추가)]
----------------------------------------------
차트 상단 '지표+' 버튼으로 원하는 지표를 추가합니다.

▶ 오버레이 지표 (차트 위에 표시)
  - SMA    : 단순 이동평균
  - EMA    : 지수 이동평균 (최근 가격 가중)
  - WMA    : 가중 이동평균
  - Hull MA: 지연이 가장 적은 이동평균 (알람 기능과 연동)
  - 볼린저밴드 (BB): 이평 ± 표준편차 밴드
  - 거래량 (Volume): 봉별 거래량 막대

▶ 오실레이터 지표 (차트 아래 별도 패널에 표시)
  - RSI        : 0~100, 70↑ 과매수 / 30↓ 과매도
  - MACD       : 단기/장기 EMA 차이, 시그널 교차 확인
  - 스토캐스틱  : 고가/저가 대비 현재 위치 (K/D선)
  - ATR        : 평균 진폭 (변동성 크기)

- 지표 이름을 클릭하면 기간·색상·굵기 등 세부 설정 가능.
- 지표 패널을 차트 안에서 드래그해 위치 이동 가능.
- 여러 지표를 동시에 추가할 수 있습니다.

----------------------------------------------
[알람 기능 (Hull MA 근접 알람)]
----------------------------------------------
- 차트 상단 '알람' 버튼으로 설정 패널을 엽니다.
- Hull MA 지표를 먼저 추가해야 알람이 동작합니다.

설정 항목:
  · 근접 거리 (USDT): 현재가와 HMA 가격 차이가 이 값 이하가 되면 발동
    예) 200 입력 → $200 이내로 접근 시 알람
  · 알람 횟수: 설정 횟수만큼 울리면 자동으로 꺼짐 (기본 1회)
  · 알람 방향: 양방향 / 위에서 접근 / 아래에서 접근 중 선택

- 알람이 울린 후 가격이 범위를 벗어났다가 다시 진입해야 재발동합니다.
- 페이지 로드 후 아무 버튼이나 한 번 클릭해야 소리가 활성화됩니다.
  (브라우저 정책)

----------------------------------------------
[소리 효과음]
----------------------------------------------
- 주문 체결 시: 짧은 비프음
- TP 도달 시 : 동전 땡그랑 소리
- SL 도달 시 : 크레이지 아케이드 패배음 (하강 멜로디)
- 강제 청산 시: 경고음 3단
- 차트 상단 소리 ON/OFF 버튼으로 모든 효과음을 끌 수 있습니다.

----------------------------------------------
[그리기 도구]
----------------------------------------------
- 차트 왼쪽 툴바에서 그리기 도구를 선택합니다.

종류:
  - 추세선    : 두 점을 잇는 직선
  - 채널      : 평행한 두 선으로 가격 범위 표시
  - 화살표    : 방향 표시
  - 텍스트    : 차트 위 메모 입력
  - 피보나치  : 되돌림 비율 자동 표시 (0~1 구간)
  - 롱/숏 도구: 진입·TP·SL 구간을 시각적으로 표시

기능:
  - 그린 도형을 클릭하면 선택 → 드래그로 이동, 핸들로 크기 조절
  - 색상·굵기 변경 가능
  - 자석 기능: 가까운 봉의 고가·저가·종가에 자동으로 달라붙음
  - 터치(모바일) 지원
  - 그리기 내용은 종목별로 자동 저장됩니다.

----------------------------------------------
[차트 화면 고점·저점 표시]
----------------------------------------------
- 화면에 보이는 봉 중 가장 높은 고가(H)와 낮은 저가(L)에
  점선으로 가격이 자동 표시됩니다.
- 스크롤(과거 조회) 시 자동으로 업데이트됩니다.

----------------------------------------------
[봉 마감 카운트다운]
----------------------------------------------
- 차트 우측 상단에 현재 봉이 마감되기까지 남은 시간이 표시됩니다.

----------------------------------------------
[회원 기능 (Firebase 클라우드 동기화)]
----------------------------------------------
- 화면 우측 상단 로그인 버튼으로 회원가입/로그인할 수 있습니다.
- 로그인하면 잔고·포지션·거래 내역·그리기가 클라우드에 저장됩니다.
  → 다른 기기에서 로그인해도 동일한 상태로 이어집니다.
- 비로그인(게스트) 상태에서는 브라우저 localStorage에만 저장됩니다.
- 프로필 설정에서 닉네임 변경·비밀번호 변경이 가능합니다.

----------------------------------------------
[랭킹 대시보드]
----------------------------------------------
- 상단 '랭킹' 버튼으로 대시보드를 엽니다. (로그인 필요)
- 탭: 통합 / 일일 / 주간 / 월간 / 종목별 랭킹
- 정렬: 총 수익 / 승률 / 거래 수 등 드롭다운으로 변경
- 컬럼: 순위·닉네임·총수익(USDT)·승률·거래수·최대수익·최대손실
- 내 순위는 ★로 강조 표시됩니다.

----------------------------------------------
[지급 자금]
----------------------------------------------
- 처음 5,000 USDT 지급.
- 강제 청산되면 안내창에서 '5,000 USDT 다시 받기'.
- 언제든 오른쪽 '계좌 초기화' 버튼으로 5,000 USDT 재지급.
  (지표·레버리지·마진 모드·알람·시간봉·그리기는 초기화되지 않습니다)

----------------------------------------------
[기준 (바이낸스 USDⓈ-M 기준)]
----------------------------------------------
- 수수료: 시장가(Taker) 0.05% / 지정가(Maker) 0.02%
- 펀딩비: 8시간마다 자동 정산
- 청산가: 바이낸스 BTCUSDT 유지증거금 브래킷 테이블 기준 계산
- 체결가: 시장가 → 현재 라스트 프라이스 / 지정가 → 설정 가격

----------------------------------------------
[자동 저장]
----------------------------------------------
- 잔고·포지션·거래 내역·그리기·지표·알람·시간봉 즐겨찾기가 자동 저장됩니다.
- 비로그인: 브라우저 localStorage (같은 브라우저에서만 유지)
- 로그인: Firebase 클라우드 (기기 무관 유지)
- 시크릿창·다른 브라우저에서 비로그인으로 열면 새로 시작됩니다.
