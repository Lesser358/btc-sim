상단 헤더

🪙 코인 선택

좌측 드롭다운에서 BTCUSDT, ETHUSDT, SOLUSDT, BNBUSDT, XRPUSDT 중 선택합니다. 변경 시 차트와 시세가 즉시 전환됩니다.

💰 실시간 가격 정보

현재가, 24시간 변동률, 24H 고/저가, 마크 가격, 펀딩비 및 남은 시간이 1초마다 갱신됩니다.

💼 총 자산

현재 잔액 + 미실현 손익을 합산한 총 자산(USDT)이 헤더 우측에 표시됩니다.

🔗 연결 상태

● 실시간 연결됨: 웹소켓 정상 / ● REST 연결: 폴링 정상 / ● 연결 오류: 네트워크 이상

차트 영역 (중앙)

  • 시간봉 버튼: 1분~1월까지 봉 종류를 선택합니다. 각 버튼 아래에 다음 봉 마감까지 남은 시간이 표시됩니다.
  • 지표+ 버튼: SMA, EMA, Hull MA, RSI 등 지표를 추가합니다.
  • 알람 버튼: Hull MA 근접 알람을 설정합니다. 활성화 시 노란색으로 표시됩니다.
  • 소리 ON/OFF: 알람 소리를 켜거나 끕니다.
  • 그리기 도구 바: 추세선, 수평선 등 차트에 직접 도형을 그립니다.
  • 차트 스크롤: 마우스 휠로 확대/축소, 드래그로 좌우 이동. 왼쪽 끝까지 스크롤하면 과거 데이터가 자동으로 추가 로드됩니다.

주문 패널 (우측)

  • 레버리지, 마진 모드, 주문 유형, 수량, TP/SL을 설정하고 Long/Short 주문을 실행합니다.
  • 하단에 계좌 현황(잔액, 미실현 손익, 총 자산, 통계)과 계좌 초기화 버튼이 있습니다.

하단 탭

  • 포지션: 현재 보유 중인 포지션 정보와 청산 조작 패널
  • 미체결 주문: 체결 대기 중인 지정가·TP/SL 주문 목록
  • 거래 내역: 완료된 모든 거래 기록 (날짜, 방향, 가격, 손익, ROE%)

📈 거래 방법 — 단계별 가이드

1
레버리지 설정
슬라이더 또는 숫자 입력창으로 2x~125x를 설정합니다. 레버리지가 높을수록 수익과 손실이 배로 커지고 청산 위험도 높아집니다.
2
마진 모드 선택
격리 투입 증거금만 위험 노출 — 설정 금액 이상은 잃지 않습니다.
교차 전체 잔액이 증거금 — 청산 저항력이 강하지만 잔액 전부를 잃을 수 있습니다.
3
주문 유형 선택
시장가 현재 가격으로 즉시 체결 (수수료 0.05%)
지정가 원하는 가격에 예약, 해당 가격 도달 시 체결 (수수료 0.02%)
4
수량 입력
증거금(USDT)을 직접 입력하거나 25% 50% 75% 100% 버튼으로 빠르게 선택합니다. 우측에 예상 계약 수량(코인)이 표시됩니다.
5
TP / SL 설정 (선택)
진입 전 익절(TP)과 손절(SL)을 미리 설정할 수 있습니다. 드롭다운으로 Price(가격), PnL(USDT 손익), ROI(% 수익률) 방식 중 하나를 선택합니다. 청산 수량도 %, USDT, 코인 단위로 설정 가능합니다.
6
주문 실행
Long 가격 상승 시 수익  |  Short 가격 하락 시 수익
💡 진입 버튼 위의 미리보기 영역에서 예상 청산가, 필요 증거금, 수수료를 확인한 뒤 주문하세요.

지정가 주문 사용 시

  • 원하는 진입 가격을 지정가 입력창에 입력한 뒤 Long/Short 버튼을 누릅니다.
  • 주문은 미체결 주문 탭에 등록되며, 시장가가 지정가에 도달하면 자동 체결됩니다.
  • 미체결 탭에서 취소 또는 TP/SL 수정이 가능합니다.

⚙️ 포지션 관리

포지션 탭 — 표시 항목

  • 방향 / 진입시간: LONG 또는 SHORT, 진입한 날짜·시각
  • 크기: 보유 중인 계약 수량 (USDT 기준)
  • 진입가 / 마크가: 평균 진입 가격과 현재 마크 가격
  • 청산가: 이 가격에 도달하면 강제청산 발생. 항상 확인 필수!
  • 미실현 손익 (PnL) / ROE%: 현재 평가 손익 금액과 증거금 대비 수익률
  • TP / SL: 설정된 TP·SL 가격. 클릭하면 수정 패널 열림

포지션 청산

부분 청산

청산 수량 입력란에 양을 입력합니다. 드롭다운으로 % / USDT / 코인 단위를 전환할 수 있습니다. 입력 후 청산 버튼 클릭.

전체 청산

전체 청산 버튼을 클릭하면 포지션 전량이 즉시 시장가로 청산됩니다.

보유 중 TP/SL 수정

포지션 탭의 TP/SL 가격을 클릭하면 수정 패널이 열립니다. 가격·PnL·ROI 방식을 변경하고 저장할 수 있습니다. 청산 수량(부분/전체)도 함께 설정 가능합니다.

⚠️ 포지션 보유 중 수량 입력창에 값을 입력하는 동안에는 차트 갱신으로 값이 덮어씌워지지 않도록 보호됩니다.

미체결 주문 탭

지정가로 예약한 주문과 TP/SL 대기 주문 목록입니다. 취소 버튼으로 주문을 삭제하거나, 수정 버튼으로 TP/SL 가격을 변경할 수 있습니다.

거래 내역 탭

완료된 모든 체결 내역이 최신순으로 표시됩니다. 날짜, 방향, 체결가, 실현 손익, ROE%를 확인할 수 있습니다. 강제청산 시 ROE%는 -100%로 고정 표시됩니다.

📊 차트 & 지표

시간봉 선택

차트 상단 버튼에서 1분, 3분, 5분, 15분, 30분, 1시간, 2시간, 4시간, 6시간, 8시간, 12시간, 1일, 3일, 1주, 1월 중 선택합니다. 각 버튼 아래에 다음 봉 마감까지 남은 시간이 실시간으로 표시됩니다.

지표 추가 방법

1
차트 상단 지표+ 버튼 클릭 → 검색창 또는 목록에서 원하는 지표 선택
2
지표 바(차트 바로 위)에 칩이 생성됨. 칩 클릭 → 기간, 색상, 굵기 설정 패널 열림
3
칩의 × 버튼으로 지표를 제거합니다. 같은 종류를 여러 개 추가할 수도 있습니다.

오버레이 지표 — 메인 차트에 선으로 표시

SMA (단순 이동평균)

설정 기간의 종가 단순 평균. 추세 방향을 파악하는 기본 지표입니다.

EMA (지수 이동평균)

최근 가격에 더 많은 가중치. SMA보다 반응이 빠릅니다.

WMA (가중 이동평균)

최근 봉에 가중치를 선형으로 적용한 이동평균입니다.

Hull MA (헐 이동평균)

지연이 가장 적은 이동평균선. 알람 기능과 연동됩니다. 추세 전환 포착에 유용합니다.

볼린저 밴드 (BB)

이동평균 ± 표준편차로 밴드 형성. 밴드 이탈 시 과매수/과매도 신호입니다.

거래량 (Vol)

봉별 거래량을 차트 하단 히스토그램으로 표시합니다.

오실레이터 지표 — 차트 하단 별도 패널

RSI

0~100 범위. 70 이상 과매수, 30 이하 과매도 신호.

MACD

단기/장기 EMA 차이. 시그널 교차 시 추세 전환 신호.

스토캐스틱

%K · %D 교차로 과매수/과매도 매매 시점 포착.

ATR

평균 진폭. 현재 시장의 변동성 크기를 수치로 측정합니다.

💡 오실레이터 지표를 추가하면 차트 아래에 별도 패널이 생기며, 메인 차트와 스크롤이 동기화됩니다.

✏️ 그리기 도구

기본 조작

이동 (기본 모드)

차트를 드래그하여 이동하거나 마우스 휠로 확대/축소합니다. 도구 선택 후 이 모드로 돌아오려면 이동 버튼을 클릭합니다.

선택 모드

도형을 클릭하여 선택 후 드래그로 이동합니다. 끝점(핸들)을 드래그하면 크기와 형태를 수정할 수 있습니다. Delete 키 또는 삭제 버튼으로 제거합니다.

🧲 자석 기능

자석 버튼(🧲)을 활성화하면 마우스가 봉의 고가(H), 저가(L), 시가(O), 종가(C) 위치에 자동으로 스냅됩니다. 정확한 가격 포인트에 도형을 그릴 때 매우 유용합니다.

그리기 도구 목록

  • 추세선: 두 점을 클릭·드래그하여 추세선을 그립니다.
  • 수평선: 클릭한 가격에 전체 구간에 걸친 수평 기준선을 그립니다.
  • 수직선: 클릭한 시각에 수직선을 그립니다.
  • 사각형: 드래그하여 특정 가격·시간 구간을 사각형으로 표시합니다.
  • 피보나치: 드래그하여 피보나치 되돌림 레벨(23.6%, 38.2%, 61.8% 등)을 표시합니다.
  • 채널: 1단계 기준선 드래그 → 2단계 클릭으로 너비 확정. 평행채널이 완성됩니다.
  • 화살표: 드래그하여 끝점에 화살촉이 있는 선을 그립니다.
  • 텍스트: 클릭 후 입력창에 텍스트를 입력합니다. 더블클릭하면 재편집할 수 있습니다.
  • Long / Short 도구: 진입가→목표가로 드래그하면 손익 구역과 R/R 비율을 시각화합니다.

색상 & 굵기 설정

그리기 바 우측의 색상 선택기와 굵기 입력창으로 선 스타일을 설정합니다. 도형을 선택한 상태에서 변경하면 선택된 도형에 즉시 적용됩니다.

💡 그리기 도형은 코인(심볼)별로 자동 저장됩니다. 새로고침 후에도 그대로 유지됩니다.

🔔 알람 기능

Hull MA(헐 이동평균선) 근처에 현재 가격이 도달하면 소리와 화면 팝업으로 알려주는 기능입니다.

⚠️ 알람을 사용하려면 먼저 지표+에서 Hull MA를 추가해야 합니다.

알람 설정 방법

1
차트 상단의 알람 버튼 클릭 → 알람 설정 패널 열기
2
알람 활성화 토글을 ON으로 변경
3
근접 거리 (USDT): 현재가와 Hull MA의 차이가 이 값 이하가 되면 알람 발동. 예: 200 입력 시 $200 이내 진입 시 알람
4
알람 횟수: 알람이 몇 번 울린 후 자동으로 꺼질지 설정합니다 (기본값: 1회). 예: 3 입력 시 3번 울리면 자동 OFF
5
알람 방향 선택:
양방향 위·아래 어디서든 근접 시 발동
위에서 접근 가격이 HMA 위에 있을 때만
아래에서 접근 가격이 HMA 아래에 있을 때만
6
볼륨 슬라이더로 소리 크기 조절 후 저장 클릭 — 소리 테스트 버튼으로 미리 확인 가능

알람 작동 방식

실시간 감지

현재 가격이 HMA 근접 범위로 진입하는 순간 발동됩니다. 범위 안에 머무는 동안은 재발동하지 않으며, 범위 밖으로 나갔다가 다시 들어올 때 다시 울립니다.

자동 비활성화

설정한 횟수만큼 울리면 알람이 자동으로 꺼집니다. 버튼이 회색으로 변하며, 다시 켜려면 알람 패널에서 재활성화하면 됩니다.

💡 브라우저 정책상 페이지 로드 후 아무 버튼이나 한 번 클릭해야 소리 알람이 활성화됩니다. 소리가 안 나면 소리 테스트 버튼을 눌러보세요.

ℹ️ 수수료 & 거래 규칙

수수료 구조

시장가 수수료 (Taker)

진입·청산 각각 0.05% 부과. 실제 바이낸스 Taker 수수료 기준입니다.

지정가 수수료 (Maker)

지정가 체결 시 0.02% 부과. 시장가보다 저렴하여 비용 절감에 유리합니다.

펀딩비 (Funding Fee)

실제 바이낸스와 동일하게 매 8시간마다 (UTC 00:00 · 08:00 · 16:00) 자동 정산됩니다. 포지션 보유 중에는 펀딩비가 잔액에서 자동 가감되므로 장기 보유 시 주의가 필요합니다. 헤더의 펀딩 / 남은시간에서 현재 율과 다음 정산까지의 시간을 확인할 수 있습니다.

강제청산 (Liquidation)

포지션의 미실현 손실이 증거금에 근접하면 강제청산이 발생합니다. 청산가는 실제 바이낸스 BTCUSDT 유지증거금 브래킷 테이블을 기반으로 계산됩니다.

⚠️ 강제청산 발생 시 해당 포지션의 증거금 전액을 잃게 됩니다. 포지션 탭의 청산가를 항상 확인하고 적절한 레버리지를 사용하세요.

계좌 초기화

주문 패널 하단의 계좌 초기화 버튼을 클릭하면 잔액이 5,000 USDT로 재설정됩니다. 지표 설정, 레버리지, 마진 모드, 알람, 시간봉, 그리기 도형은 초기화되지 않고 유지됩니다.

실시간 데이터 출처
